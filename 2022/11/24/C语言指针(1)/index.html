# 一。指针简介

指针是 C 语言的灵魂，它不仅仅是 C 语言中的一个重要概念及其特点，也是 C 语言比较困难的部分。指针也就是 内存地址 ，指针变量是用来存放内存地址的变量，不同类型的指针变量所占用的存储单元长度是相同的 (毕竟只是一个地址)，而存放数据的变量因数据的类型不同，所占用的存储空间长度也不同。【其实就像酒店的房间，房间的 门牌号 (指针 \ 内存地址) 可以都是三位数 如 803 . 012，但是 房间类型 (不同数据类型的储存空间长度) 可以随着住户的需求而改变，像 单人间 ， 双人间 ，杂物间？】

合理的使用指针可以提高效率。

合理的使用指针可以提高灵活度。

不合理的使用指针就会大 (吉) 寄大 (利) 历；

因为指针是一把双刃剑，如果对指针不能正确理解和灵活有效的应用，利用指针编写的程序也很容易发生各式各样的错误，同时程序的可读性也会大打折扣。所以指针的使用，不是 吉，就是 寄。至于是吉还是寄，就看诸位的功底了。}

# 1. 内存

C 语言所有的变量都存放在内存中，每个人都有它的归宿， 而各种变量如 int ,char … ，它们的归宿就是内存，总之嘛，内存作为容器用来存放变量。

# 2. 内存地址

内存地址就好比一个人的身份证，每个人的身份证都是唯一的，内存地址也是唯一的，可以通过唯一的内存地址，找到唯一的内存数据。
